# 项目知识库

> 永久记忆，跨会话沉淀。重要发现写入本文件。
> 与 progress.md 的分工：progress.md 是会话级临时日志，本文件是项目级长期知识。
> 运维框架特有：故障结论必须写入本文件，避免重复故障。
> 知识库是 Loop 复利学习的基础——每轮部署/故障/优化的结论沉淀于此，下一轮调用。
